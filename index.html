Sobre Mim

Olá! Sou João Pedro Santana, desenvolvedor e estudante de Análise e Desenvolvimento de Sistemas.

Sou formado em Ciências Contábeis, o que me proporcionou uma visão analítica e estratégica, mas foi na programação que encontrei minha verdadeira paixão.

Atualmente, estou aprimorando minhas habilidades em C#, com foco no desenvolvimento de aplicações web e automação de processos. Também possuo experiência com Java e Python, que utilizei em projetos acadêmicos e pessoais, incluindo ferramentas para análise de dados financeiros e conversão de arquivos XML.

Se quiser conhecer mais sobre meus projetos, fique à vontade para explorar meu portfólio!

Projetos

Conversor XML NF-e para Excel

Ferramenta para converter arquivos XML de Nota Fiscal Eletrônica em planilhas Excel. O usuário anexa os arquivos xml de notas fiscais e envia para a API que extrai os dados, converte e os organiza em uma única tabela Excel formato .xlsx.

Tecnologias: C#, ASP.NET, JavaScript, HTML, CSS.
